# clj-telnet

A Clojure library designed to test a WS socket connection

## Usage

**NOTE**: Requires [JDK](https://openjdk.java.net/) and [leiningen](https://leiningen.org/) to be installed!

### Build
```
lein do clean, install
```
### Test
```
lein test
```
### Run
```
lein trampoline run
```
